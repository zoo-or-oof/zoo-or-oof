## 🔧 Common Issues and Fixes

**Script doesn't execute / nothing happens**
- Ensure your executor is fully attached to the Roblox process before executing
- Try re-injecting the executor, then re-pasting the script

**GUI doesn't appear in-game**
- Some GUIs load with a delay; wait 5–10 seconds after execution
- Check if your executor supports `Drawing` library calls

**Auto-farm stops mid-session**
- This is usually caused by a map update resetting spawn points; restart the script
- Ensure Anti-AFK is toggled ON

**ESP shows incorrect player positions**
- This occurs after game updates that change character model paths; wait for a patch update

**Game crashes after injection**
- Lower your Roblox graphics settings to reduce memory load
- Disable aimbot assist if using a low-RAM device

**Teleport sends character to wrong location**
- Coordinate maps reset on game updates; update to the latest script version
